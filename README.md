# Dotfiles
![My terminal](resources/images/dotfiles.png)

## Alacritty
```shell
ln -s ~/.config/dotfiles/alacritty.yml ~/.config/alacritty/alacritty.yml
```

## Tmux
```shell
ln -s ~/.config/dotfiles/.tmux.conf ~/.tmux.conf
```

## zsh
```shell
ln -s ~/.config/dotfiles/ej.zsh-theme ~/.oh-my-zsh/custom/themes/ej.zsh-theme

ln -s ~/.config/dotfiles/.zshrc ~/.zshrc
```
